# Malicious Chrome Extensions ID Repository

This repository contains IDs of malicious extensions that have been declared or are considered potentially harmful to the company's internal security (e.g., VPN extensions, data-stealing extensions, etc.).

The file **Malicious_Chrome_Extensions_ID.txt** provides a list of IDs that can be used to create KQL or other types of queries in your defense software.
